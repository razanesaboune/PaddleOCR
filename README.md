# French Version
# PaddleOCR - Extraction de texte pour les cartes d'identité et passeports libanais

Ce projet utilise **PaddleOCR** pour extraire automatiquement le texte présent sur les cartes d’identité et les passeports libanais.  
L’objectif est d’obtenir une reconnaissance précise, notamment des **chiffres arabo-indiens**, noms, dates et numéros d’identification.

# Fonctionnalités
- Extraction du texte en **arabe** et **anglais**  
- Reconnaissance des chiffres arabo-indiens (٠١٢٣٤٥٦٧٨٩)  
- Support pour cartes d’identité libanaises et passeports  
- Possibilité de fine-tuning du modèle pour améliorer la précision  
- Prétraitement automatique des images (redimensionnement, rotation, etc.)

# English Version
# PaddleOCR - Text Extraction for Lebanese IDs and Passports
This project uses **PaddleOCR** to automatically extract text from Lebanese identity cards and passports.  
The goal is to achieve high accuracy, especially for **Arabic text** and **Arabic-Indic digits**, including full ID numbers.

## Features
- OCR for **Arabic** and **English** text  
- Recognition of Arabic-Indic digits (٠١٢٣٤٥٦٧٨٩)  
- Supports Lebanese ID cards and passports  
- Image preprocessing (resize, rotation correction, denoising)  
- Optional model fine-tuning to improve accuracy
